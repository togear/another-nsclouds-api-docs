# DashScope - Text Completions

### 1. Overview

DashScope's text completions capability.

{% hint style="success" %}
This API is compatible with the OpenAI Completions interface.
{% endhint %}

**Supported models：**

* `MiniMax-M2.5`
* `MiniMax-M2.7`
* `qwen3.6-flash`
* `qwen3.6-max-preview`
* `qwen3.6-plus`
