# Anthropic - Text Completions

### 1. Overview

Anthropic's text completions capability.

{% hint style="success" %}
This API is compatible with the OpenAI Completions interface.
{% endhint %}

**Supported models：**

* `claude-haiku-4-5-20251001`
* `claude-haiku-4.5`
* `claude-opus-4.5`
* `claude-opus-4.6`
* `claude-opus-4.7`
* `claude-sonnet-4.5`
* `claude-sonnet-4.6`
