# OpenAI - Text Completions

### 1. Overview

OpenAI's text completions capability.

{% hint style="success" %}
This API is compatible with the OpenAI Completions interface.
{% endhint %}

**Supported models：**

* `gpt-5`
* `gpt-5.1`
* `gpt-5.2`
* `gpt-5.3-codex`
* `gpt-5.4`
* `gpt-5.4-mini`
* `gpt-5.5`
