# DashScope - Chat Completions

### 1. Overview

DashScope's conversation API capability.

{% hint style="success" %}
This API is compatible with the OpenAI Chat Completions interface.
{% endhint %}

**Supported models：**

* `MiniMax-M2.5`
* `MiniMax-M2.7`
* `qwen3.5-flash`
* `qwen3.5-plus`
* `qwen3.6-flash`
* `qwen3.6-max-preview`
* `qwen3.6-plus`
