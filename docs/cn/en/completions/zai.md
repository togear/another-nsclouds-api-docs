# Z.AI - Text Completions

### 1. Overview

Z.AI's text completions capability.

{% hint style="success" %}
This API is compatible with the OpenAI Completions interface.
{% endhint %}

**Supported models：**

* `glm-4.6`
* `glm-4.6v`
* `glm-4.7`
* `glm-5`
* `glm-5.1`
