# DashScope - Text Completions

### 1. Overview

DashScope's text completions capability.

{% hint style="success" %}
This API is compatible with the OpenAI Completions interface.
{% endhint %}

**Supported models：**

* `qwen3.5-flash`
* `qwen3.5-plus`
* `qwen3.6-flash`
* `qwen3.6-max-preview`
* `qwen3.6-plus`
* `qwen3.7-max`
